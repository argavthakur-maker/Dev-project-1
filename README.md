# 🏠 Hosteler's Expense Manager

A modern full-stack expense management platform built specifically for hostel students to track, manage, and optimize their daily spending habits.

This application helps students monitor expenses, maintain budgets, organize spending categories, and gain better control over their finances through an intuitive and responsive dashboard.

## ✨ Features

* 💸 Add and manage expenses in real time
* 🗑️ Delete expenses instantly
* 📊 Interactive spending dashboard
* 📁 Category-based expense organization
* 💰 Budget monitoring system
* 🔄 Django REST API integration
* 🎨 Modern premium dark-themed UI
* 📱 Responsive design for desktop and mobile
* ⚡ Fast frontend powered by React + Vite
* 🗄️ Secure backend powered by Django
